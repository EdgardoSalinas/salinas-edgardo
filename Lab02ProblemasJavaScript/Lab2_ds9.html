¿Qué es Javascript y en qué se asemeja a Java? JavaScript es un lenguaje de programación utilizado para crear y controlar contenido dinámico del sitio web (es decir, cualquier cosa que se mueva, refresque o cambie en pantalla sin volver a cargar una página web). Funciones de JavaScript: Filas de diapositivas Animaciones Autocompletar sugerencias de texto Formularios interactivos JavaScript permite que tu página web actualice tus datos de Facebook y Twitter en tiempo real. La principal diferencia es: Java es un lenguaje que hay que compilar. Es necesario utilizar la JVM (Java Virtual Machine) para que el código pueda ser interpretado por la «máquina». JavaScript es un lenguaje interpretado porque se lee y traduce directamente a medida que se ejecuta el programa.
